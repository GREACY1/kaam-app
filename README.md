# KaamConnect

KaamConnect is a **hyperlocal gig worker platform** that connects customers with nearby skilled workers such as electricians, plumbers, carpenters, delivery helpers, labourers etc.

## Features

* Worker registration with skills
* Customers can post jobs
* Location based job matching
* Real-time communication using Socket.io

## Tech Stack

Frontend

* React
* TypeScript

Backend

* Node.js
* Express.js
* MongoDB
* Socket.io

## Installation

Clone the repository

git clone https://github.com/GREACY1/kaam-app.git

Backend setup

cd backend
npm install
node index.js

Frontend setup

cd frontend/kaamconnect-app
npm install
npm start
